## ‚ú® Features

I tried to pack in as many modern web features as I could:

* **Glassmorphism Nav:** A sticky, blurred navigation bar that stays with you as you scroll.
* **Smooth Scrolling:** Clicking a nav link will glide you smoothly to that section.
* **Dropdown Menus:** Clean, CSS-only dropdowns for a better user experience.
* **Stylish Hero:** A big, bold hero section to grab your attention.
* **CSS Grid Layout:** A responsive grid for all the anime cards.
* **Cool Hover Effects:** Check out the anime cards and buttons! They lift up and glow when you hover over them.
* **Fully Responsive:** (Well, I'm trying!) It should look good on mobile devices, too.
